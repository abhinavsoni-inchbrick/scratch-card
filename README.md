# Inch & Brick — Scratch Card

Interactive scratch card revealing the winner **DARAAB**.

## Live Site

**https://abhinavsoni-inchbrick.github.io/scratch-card/**

## GitHub Pages Setup (one-time)

1. Open [Repository Settings → Pages](https://github.com/abhinavsoni-inchbrick/scratch-card/settings/pages)
2. **Build and deployment → Source:** choose **Deploy from a branch**
3. **Branch:** `main` · **Folder:** `/ (root)`
4. Click **Save**
5. Wait 1–2 minutes, then open the live URL with hard refresh (`Ctrl+F5`)

## Features

- Black & gold Inch & Brick branding
- Scratch to reveal **DARAAB** at 60%
- Colorful party popper + gold sparkles + celebration sound
- Mobile and desktop support
